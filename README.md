## Основные функции

- Отображение древовидной структуры отделов.
- Просмотр списка сотрудников в каждом отделе.
- Добавление новых сотрудников через веб-интерфейс.
- Подробная информация о каждом сотруднике.

## Установка и запуск

1. **Клонируйте репозиторий:**
   ```bash
   git clone <ссылка на репозиторий>
   cd company_structure


2. **Установите зависимости:**
    pip install -r requirements.txt


3. **Настройте базу данных:**
    Убедитесь, что PostgreSQL установлен и запущен.
    Создайте базу данных company_structure.
    Обновите настройки базы данных в settings.py:

    DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.postgresql',
        'NAME': 'company_structure',
        'USER': 'your_username',
        'PASSWORD': 'your_password',
        'HOST': 'localhost',
        'PORT': '5432',
        }
    }

4. **Примените миграции:**
    python manage.py migrate

5. **Загрузите фикстуры (по желанию), или используйте свои данные:**
    python manage.py loaddata departments.json
    python manage.py loaddata employees.json

6. **Создайте суперпользователя:**
    python manage.py createsuperuser

    Логин: root
    Пароль: root

7. **Запустите сервер:**
    python manage.py runserver

8. **Откройте в браузере:**
    http://127.0.0.1:8000/


Использование:
    Главная страница: Отображает список главных отделов. Каждый отдел кликабелен, что позволяет перейти к его подразделениям и сотрудникам.

    Добавление сотрудника: Нажмите кнопку "Добавить сотрудника" на главной странице, чтобы открыть форму для добавления нового сотрудника.

    Администрирование: Перейдите по кнопке "Администрирование" для доступа к панели администратора Django.

Фикстуры:
    В папке departments/fixtures/ находятся файлы departments.json и employees.json, которые содержат примеры данных для отображения структуры отделов и сотрудников.
